# 비즈니스 관측

비즈니스 관측 화면은 인증, 대기열, 좌석 Hold, 주문 결제, 봇 탐지 결과를 한 화면에서 확인하는 기준입니다. 티켓팅 흐름이 정상적으로 이어지는지와 전환 구간에서의 병목을 함께 확인합니다.

---

## 주요 대시보드

| 대시보드 | 주요 지표 | 목적 |
|---|---|---|
| **비즈니스 KPI** | 인증 성공률, 대기열 상태, Hold 성공률, 주문 결제 전환 | 티켓팅 흐름과 전환 지표 확인 |
| **프론트엔드 행동 분석** | 사용자 유입, 주요 행동 이벤트, 화면 전환 흐름 | 사용자 행동과 화면 흐름 확인 |

---

## 비즈니스 KPI

인증, 대기열, 좌석 Hold, 주문 결제, 봇 탐지 결과를 한 화면에서 확인해 티켓팅 흐름과 전환 지표를 같이 확인합니다.

## 운영 기준

| 구분 | 확인 지표 | 목적 |
|---|---|---|
| **인증** | 인증 시도, 성공, 성공률 | 로그인 구간 정상 여부 확인 |
| **대기열** | 대기 인원, 대기 상태, 대기 시간 | 예매 오픈 구간 적체 확인 |
| **좌석 Hold** | Hold 시도, 성공, 실패, 만료 | 좌석 선점 흐름과 실패 원인 확인 |
| **주문 / 결제** | 주문 건수, 결제 시도, 성공, 실패, 전환율 | 전환 병목과 실패 지점 확인 |
